# 小组分工

当前使用角色制，不写真名。

## 组长

你当前承担：复现 + 系统 + 集成。负责 GitHub 仓库、合并 PR、把 VDT 复现结果接入项目、最终统一报告/PPT/demo。

## 复现负责人

负责 VDT 官方环境、NewsCLIPpings/VisualNews 数据状态、训练配置记录、指标表。交付：`docs/reproduction_logs/YYYYMMDD_vdt_run.md`、结果 CSV、混淆矩阵。

## 系统负责人

负责 Gradio demo、统一输出 schema、heuristic/VDT/E3-VDT 三种后端、答辩展示样例。交付：`demo/`、`src/e3vdt/inference/`、`examples/demo_cases.jsonl`。

## 报告负责人

负责结课报告、答辩 PPT、结果图表、成员贡献表。交付：`docs/report/final_report.docx`、`docs/ppt/final_presentation.pptx`。
